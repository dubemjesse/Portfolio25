IFRS Reports App

A full-stack financial reporting and data explorer application designed for IFRS-compliant datasets.

Tech Stack

  • Frontend: React, TypeScript, Vite, Material UI
  • Backend: Node.js, Express, TypeScript
  • Database: Microsoft SQL Server (msnodesqlv8)

Core Features

  • Secure user authentication with JWT
  • Configurable sidebar for navigating database-backed IFRS reports
  • Dynamic data tables with server-side pagination, sorting, and search
  • Clean UI with uppercase headers and date formatting for financial data
  • Modular architecture with separate backend and frontend layers

Highlights

  • Built for scalability and maintainability with clear project structure
  • Supports custom table configurations for flexible reporting
  • Production-ready build processes for both backend and frontend

CoinFlow — Personal Expense & Reports Tracker

Overview: CoinFlow is a modern, responsive web app built to help users track daily expenses, monitor budgets, and visualize spending trends through interactive reports and charts.

Key Features

  • Add, categorize, and manage expenses with visual budget progress.
  • Generate dynamic spending reports with date and category filters.
  • View trends via a custom Canvas-based chart (daily, weekly, monthly).
  • Manage simple financial reminders.
  • Data persists locally using localStorage, with CSV export support.

Tech Stack

  • Frontend: HTML5, CSS3 (Flexbox & Grid), JavaScript
  • Charting: Chart.js for dynamic, interactive visualizations
  • Icons: Font Awesome for scalable vector icons

Structure

Includes dedicated pages for Dashboard, Transactions, Reports, and Reminders — all unified by consistent styling and intuitive navigation.

Highlights

  • Custom-built chart rendering for performance and flexibility.
  • Fully client-side with demo data seeding for instant preview.
  • Clean UI emphasizing simplicity and usability.
